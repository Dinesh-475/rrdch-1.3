# RRDC Web Portal

This project is the web portal for the Rajarajeswari Dental College & Hospital (RRDCH). 

## Features
- **Public Website:** Provides information about admissions, departments, doctors, and patient services.
- **Chatbot / AI Assistant:** Integrated chat widget available across pages.
- **Student & Doctor Portals:** Dedicated portal interfaces.
- **Backend API Integration:** Serverless API logic (`/api`) and Supabase integration (`/supabase`) for managing admissions and enquiries.

## Structure
- `index.html`, `admissions.html`, `patients.html`... - Main frontend pages
- `/css` - Styling and themes
- `/js` - Frontend logic and AI integration
- `/api` - Backend logic and API routes
- `/images` & `/pictures` - Website assets
